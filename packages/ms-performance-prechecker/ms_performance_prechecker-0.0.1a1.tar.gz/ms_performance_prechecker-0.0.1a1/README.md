# MindStudio Performance Prechecker

## 介绍
- 执行 `ms_performance_prechecker`
  ```sh
  ms_performance_prechecker
  ```
- 参数

  | 参数                 | 说明                                                            |
  | -------------------- | --------------------------------------------------------------- |
  | -t, --check_type  | 检查项类型，可选值：basic, deepseek                                  |
